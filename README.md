# Blog Application (Django)

## 📌 Description
A full-stack blog application built with Django that allows users to create, read, update, and delete blog posts.
The project demonstrates backend development, database management, and clean project structure.

## 🛠 Tech Stack
- Python
- Django
- SQLite (development)
- HTML / CSS
- Git & GitHub

## 🚀 Features
- User authentication
- Create, edit, and delete blog posts
- Admin dashboard
- Secure backend logic
- Clean MVC architecture
